CSS

Cascading Style Sheets의미합니다.

Cascading : 계단식
Style : 멋을 내다
Sheets : (종이) 한 장
즉 계단식으로 스타일을 정의하는 문서
HTML 문서에 스타일을 정의해주는 언어 이고
HTML 문서는 태그가 태그를 포함하는 계단식 구조를 지니고 있어 CSS는 계단식 특성에 맞게 스타일을 꾸며주는 이름을 정의하고 있습니다.

HTML : 웹 문서의 콘텐츠를 구성하는 언어 마크업 언어
CSS : 웹 문서의 콘텐츠에 스타일을 추가하는 언어(색상, 크기, 위치 등) 스타일 시트 언어.

CSS 기본 문법

구성 요소
p {
  color: red; /*p 태그에 color를 red로 정의합니다.*/
   }
p (선택자) : 어떤 요소에 스타일을 적용 할지에 대한 정보, 예 : p태그에 스타일을 적용하겠다
{} : 선택한 요소에 적용할 스타일을 정의하는 영역
color (속성명) : 어떤 스타일을 정의하고 싶은지에 대한 정보(색상, 크기 등), 예 : color를 정의하겠다
red (속성값) : 어떻게 정의하고 싶은지에 대한 정보, 예 : red로 정의하겠다
/**/ (CSS주석) : /**/ html주석과 동일하게 쓰이며 /*형태의 주석은 다른 언어에서도 많이 사용됩니다.

태그안에 직접적으로 스타일을 정의하는 방식 나중에 웹 콘텐츠와 스타일시트를 분리하기 위해서는 사용하지 않는 편이 좋습니다.

사용방법

페이지 소스 확인하기

태그 안에 직접적으로 스타일을 적용할 수 있습니다.

  • 인라인 스타일 : 태그에 직접 기술하기

HTML 문서에 style태그를 추가하여 그안에 CSS 코드를 작성할 수 있습니다.

  • 스타일 태그 : 스타일시트를 위한 태그를 추가하여 기술하기

새롭게 *.CSS인 스타일시트 파일을 생성해 CSS 코드를 작성하고 HTML문서에 연결해 줄 수 있습니다. 이때 link 태그를 이용해서 연결해 줍니다.

  • 문서 간의 연결 : 스타일시트 문서를 따로 작성하여 HTML 문서와 연결하기

마지막으로 한 예시처럼 분리하여 작업하는 것이 바람직합니다.

선택자의 종류

  • 기본 선택자
  • 그룹 선택자
  • 특성 선택자
  • 결합 선택자
  • 의사 클래스
  • 의사 요소

기본 선태자 & 그룹 선택자

*{ color: blue; } // 전체 선택자 : *(애스터리스크)는 '문서 내의 모든 요소'를 의미하는 기호입니다.
p{ color: blue; } // 태그 선택자 : 주어진 이름을 가진 요소를 선택합니다. '유형 선택자'라고도 합니다. 해당이름의 요소가 다수일 경우, 요소 모두가 선택됩니다.
.text{ color: blue; } // 클래스 선택자 : 해당 class 속성값을 가진 요소를 선택하고 요소가 다수일 경우 해당 요소를 모두 선택합니다.
#topic{ color: blue; } // 아이디 선택자 : id 속성값을 가진 요소를 선택합니다. id는 고유한(unique) 식별자 역할을 하는 전역 속성으로 하나의 값만을 정의하여 사용해야 합니다. 그래서 하나의 요소를 선택하여 꾸며줄때 이용합니다.
h1, p, div{ color: blue; } // 그룹 선택자 : 다양한 유형의 요소를 한꺼번에 선택하고 할 때 사용됩니다. 쉽표(,)를 이용해 선택자를 그룹화합니다. 한번에 처리할 스타일을 지정할 때 이용합니다.

선택자가 겹치는 경우

기본적으로 나중에 작성된 스타일이 적용됩니다. 선택자가 다르지만 요소가 겹치는 경우, 선택자 우선순위에 의해 적용될 스타일이 결정됩니다.
div id="id" class="class"문서 간의 연결 : 스타일시트 문서를 따로 작성하여 HTML 문서와 연결하기/div
#id{ color: blue; }
.class{ color: red; }
div{ color: green; }
아이디 선택자 > 클래스 선택자 > 태그 선택자 #id{ color: blue; } 이 적용됩니다.

텍스트 관련 속성

  • font-family : 글꼴을 정의합니다.
    *{
    font-family: Times, monospace, serif;
    // 이런식으로 글꼴을 연달아 기입할 수 있습니다. Times폰트를 우선으로 하고 폰트가 없거나 문제가 있을시 다음 폰트 monospace로 맟추어 폰트를 지정해 줍니다.
    }

  • font-size : 수치와 단위를 지정해 글자 크기를 정의합니다.
    span{ font-size: 16px; } px : 모니터 상의 화소 하나 크기에 대응하는 절대적인크기
    span{ font-size: 2rem; } rem : html 태그의 font-size에 대응하는 상대적인 크기 최상위 요소(html)의 폰트사이즈로 부터 몇배수 만큼에 크기를 지정하는 단위입니다.
    span{ font-size: 1.5em; } em : 부모태그(상위태그)의 font-size에 대응하는 상대적인 크기 이것도 위에 상위태그의 폰트사이즈로 부터 몇배수 만큼의 크기를 지정하는 단위입니다.

  • text-align : 블록 내에서 텍스트의 정렬 방식을 정의합니다. 미리 정의된 키워드 값을 지정합니다.
    p{ text-align: left; or right } left/right : 왼쪽 또는 오른쪽 정렬합니다.
    p{ text-align: center; } center : 가운데로 정렬합니다.
    p{ text-align: justify; } justify : 양끝 정렬합니다.(마지막 줄 제외)

  • color : 텍스트의 색상을 정의합니다. 다양한 방법으로 색상을 지정 할 수 있습니다.
    p{ color: red; } 키워드 : 미리 정의된 색상별 키워드를 사용하여 간단하게 나타낼 수 있습니다.
    p{ color: #FF0000; } RGB 색상 코드 : # + 여섯자리 16진수 값 형태로 지정할 수 있습니다.
    p{ color: rgb(100%, 0%, 0%); } RGB 함수 : Red, Green, Blue의 수준을 각각 정의해 지정할 수 있습니다.

인라인 이냐 or 블록이냐

display

요소를 블록과 인라인 요소 중 어느 쪽으로 처리할지 정의합니다.

(예)
div{ display: inline;} // 블록 레벨 요소인 div를 인라인으로 처리할 수 있습니다.
a{ display: block;} // 인라인 요소인 a를 블록 레벨로 처리할 수 있습니다.

display 속성값
  • inline : 인라인으로 처리한다.
  • block : 블록 레벨로 처리한다.
  • inline-block : 인라인으로 배치하되, 블록 레벨 요소의 속성을 추가할 수 있도록 처리한다.
  • none : 디스플레이(표시)하지 않는다.

border

요소가 차지하고 있는 영역에 테두리를 그릴 수 있습니다.
속성값에는 테두리의 두꼐, 모양, 크기 등을 지정할 수 있는데 이러한 속성을
'단축속성'이라고 합니다.

(예)
span{ border: 2px solid green; } // 스판테그 요소에 두께가 2px인 직선 모양(solid)의 초록(green) 테두리를 만들어줘!

border 속성의 하위 속성
border속성에 지정하는 값들을 따로따로 지정할 수도 있습니다.
  • border-color : color 정의 방식과 동일
  • border-width : thin, medium, thick 등의 키워드 또는 px, em, rem 등의 단위를 사용합니다. 두께를 정의 할 때 이용합니다.
  • border-style : none(기본값), solid(직선), dotted(점선), dashed(긴 점선) 등 스타일을 정할 수 있습니다.

박스 모델

브라우저가 요소를 렌더링 할 때, 각각의 요소는 기본적으로 사각형 형태로 영역을 차지 하게 됩니다.
이영역을 '박스'라 표현하며, CSS는 박스의 크기, 위치, 속성(색, 배경, 테두리 모양 등)을 결정할 수 있습니다.

박스 모델의 구성

  • 콘텐츠 영역 : width, height
  • 안쪽 여백 : padding 안쪽여백을 만들어 콘텐츠영역의 위치를 조정할 수 있습니다.
  • 바깥쪽 여백 : margin 바깥쪽 여백을 만들어 다른 요소영역과의 거리를 늘려 전체적으로 위치를 조정할 수 있습니다.
  • 경계선(테두리) : border-width

padding & margin

요소에 테두리를 기준으로 안쪽의 여백을 padding이라고 말하고 테두리 바깥쪽을 margin이라고 합니다.

padding 하위 속성
padding-top, padding-left, padding-right, padding-bottom

margin 하위 속성
margin-top, margin-left, margin-right, margin-bottom

각 방향에 따라 여백의 크기를 지정해 줄 수 있습니다. 여러 값 한번에 정의하기
span{
     padding: 10px, 20px, 30px, 40px;
     margin: 40px, 30px, 10px, 20px;
     // 위, 오른쪽, 아래쪽, 왼쪽 순서대로 방향을 지정하여 값을 지정해 줍니다.
}

box-sizing
요소의 너비(width)와 높이(height)를 계산하는 방법을 지정해 사용합니다.
  • content-box : box-sizing속성의 기본값으로 따로 content-box을 선언해 주지 않아도 content-box의 너비와 높이가 콘텐츠 영역만을 포함합니다.
    요소의 크기와 상관없이 콘텐츠의 크기를 조정하고 싶을때 사용할 수 있겠습니다.
  • border-box : 너비와 높이가 안쪽 여백과 테두리까지 포함합니다.
    요소의 크기를 지정할때는 이 형식을 이용해 요소의 크기만을 지정해 줄수 있습니다.

background

배경(background)은 콘텐츠의 배경을 정의합니다.
단축 속성으로써 색상, 이미지, 반복 등 다양한 하위 속성을 정의할 수 있습니다.
  • background-color : 배경 색을 정의합니다.
    background-color: red;

  • background-image : 배경 이미지를 정의합니다.
    background-image: url(사진 주소);

  • background-position : 배경 이미지의 초기 위치를 정의합니다.
    background-position: center // left right center top bottom 등 위치를 지정 여러개를 사용해서 위치를 정의 할 수도 있습니다.

  • background-size : 배경 이미지의 크기를 정의합니다.
    background-size: cover;
    // cover : 요소의 사이즈에 맞게 깨지지 않을정도로 크기를 지정하여,
    contain : 요소에 맞게 지정하나 이미지가 잘리지 않고 깨지지않는 선에서 크기를 맟춤
    여러 키워드나 직접적으로 300px, 200px로 값을 줄 수 있습니다.

  • background-repeat : 배경 이미지의 반복 방법을 정의합니다. 반복을 할지 말지를 정할 수도 있습니다.
    background-repeat: repeat
    // repeat : 배경에 남는 부분은 자동으로 이미지를 계속 생성(반복)해서 빈 곳을 채워 줍니다.
    no-repeat : 배경에 남는 부분이 있어도 생성(반복)을 하지 않게 합니다.
    background-image: url(사진 주소) no-repeat; 이런식으로 바로 이용할 수도 있습니다.
